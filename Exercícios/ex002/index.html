Parágrafos e Quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar em especifico
como esse, você pode usar a tag <br>

Vamos adicionar alguns símbolos especiais:
® <- Marca registrada
© <- CopyRight
™ <- TradeMarck
€ <- Euro
£ <- Libra
¥ <- Yen
¢ <- Cents
Δ <- Delta
↑ ↓ ← → <- Setas

Vamos adicionar alguns emojis: → ☝ 🤓
